getting started with Jade

get from zero to a running Jade program in 5 minutes.

step 1 — get the tools

clone and build from source:

git clone https://github.com/Fundiman/JadeLang
cd JadeLang

build jscc (the compiler):

cd jade/jscc
mkdir build && cd build
meson setup ..
ninja

build jvk (the runtime):

cd ../../jvk
mkdir build && cd build
meson setup ..
ninja

requirements: g++, meson, ninja, llvm-18-dev

on Ubuntu/Debian:

sudo apt install g++ meson ninja-build llvm-18-dev

on macOS:

brew install llvm meson ninja

on Windows: use MSYS2 or WSL with the same apt commands.

step 2 — install the VS Code extension

  1. download the extension from Fundiman/jade-docs
  2. copy the jade-vscode/ folder to:

- Windows: %USERPROFILE%\.vscode\extensions\jade-lang - Linux/macOS: ~/.vscode/extensions/jade-lang

  1. restart VS Code
  2. open any .jsc file — syntax highlighting just works

step 3 — your first project

mkdir my-first-jade-app
cd my-first-jade-app
python3 path/to/JadeLang/jpkg/jpkg.py init

this creates:

my-first-jade-app/
  jade.toml
  src/
    main.jsc
  out/

step 4 — write some Jade

open src/main.jsc. it already contains:

package com.yourname.my-first-jade-app;

import jade.stdlib.io;

@entry
def main(args []str) i32 {
    io.println("hello from Jade!");
    return 0;
}

change "hello from Jade!" to whatever you want.

step 5 — compile and run

# compile
jscc src/main.jsc out/main.joc

# run
jvk out/main.joc

or use jpkg:

python3 path/to/jpkg/jpkg.py build
python3 path/to/jpkg/jpkg.py run

output:

jvk: loaded 'com.yourname.my-first-jade-app' — 2 slice(s)
jvk: runtime initialized on linux
hello from Jade!
jvk: exited with code 0

what's in the .joc?

jvk --info out/main.joc
package:  com.yourname.my-first-jade-app
slices:   2
  [x86_] 1760 bytes (native)
  [LLBC] 2020 bytes (LLVM bitcode)

your program is portable. on ARM it'll JIT the LLBC slice automatically.
